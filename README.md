# Recall

Recall is an open‑source digital signage platform designed for Raspberry Pi.

Features:
- Web dashboard
- Media upload
- Playlist editor
- Layout editor
- Device monitoring
- Screen rotation
- Player agent
- GStreamer renderer support
- Docker support

Quick install:

git clone https://github.com/YOURNAME/recall
cd recall
chmod +x recall-install.sh
./recall-install.sh
