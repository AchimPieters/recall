# Recall Enterprise v1.0.0 Release Notes

Release date: 2026-03-13

## Highlights
- Backend geconsolideerd op `backend/` met API-versioning onder `/api/v1`.
- Security-fundament: MFA setup/verify, API-key governance en audit logging.
- Operations-fundament: release gates, migration policy en DR-runbook.
- Enterprise UX baseline: dashboards voor fleet, analytics, alerts en OTA beheer.

## Breaking/important notes
- Legacy onversieerde API-routes worden niet ondersteund; gebruik `/api/v1/*`.
- Productie vereist expliciete secret management en HTTPS-instellingen.

## Validation summary
- Core backend + agent test suites groen.
- Supply-chain workflow aanwezig (SBOM, vuln scan, signing).
- Coverage gate verhoogd met gecombineerde backend+agent check.
